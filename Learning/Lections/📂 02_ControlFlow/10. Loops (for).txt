Тема №10: Цикл for — Класика програмування 👑
Цикл for — це "швейцарський ніж" розробника. Він дозволяє зібрати створення лічильника, перевірку умови та його зміну (інкремент) в одному рядку.

1. Анатомія циклу for 🏗️
Він складається з трьох частин у круглих дужках, розділених крапкою з комою (;):
Ініціалізація (let i = 0): Створюємо змінну-лічильник. Виконується лише один раз на самому початку.
Умова (i < 5): Перевіряється перед кожним колом. Якщо true — код працює, якщо false — цикл завершується.
Ітерація (i++): Зміна лічильника після кожного виконаного кола.

Приклад:
for (let i = 1; i <= 3; i++) {
  console.log(`Повторення №${i}`);
}

2. Чому саме i? 🤔
Традиційно в програмуванні лічильник називають літерою i (від слова index або iterator). 
Якщо всередині першого циклу є ще один, його називають j, потім k. Але ти можеш назвати змінну як завгодно, наприклад let step = 0.

3. Зворотний цикл (Countdown) ⏳
Цикл for може працювати і у зворотному напрямку. Це часто потрібно для таймерів або коли ми обробляємо дані з кінця до початку.
Для цього нам потрібно змінити три речі:
Початок: Стартуємо з великого числа (let i = 10).
Умова: Цикл працює, поки лічильник більший за щось (i >= 0).
Крок: Зменшуємо лічильник (i--).

4. Команда break (Стоп-кран) 🛑
Вона повністю зупиняє цикл і "викидає" програму з нього.
Коли використовувати: Коли ти шукаєш щось одне конкретне. Як тільки знайшов — навіщо крутити цикл далі?
const box = ["Книга", "Ключі", "Ручка", "Телефон"];

for (let i = 0; i < box.length; i++) {
  console.log(`Перевіряю: ${box[i]}`);
  if (box[i] === "Ключі") {
    console.log("Знайшов! Зупиняю пошук.");
    break; // Цикл зупиниться на індексі 1. "Ручку" та "Телефон" ми навіть не перевіримо.
  }
}

5. Команда continue (Пропуск) ⏭️
Вона не зупиняє весь цикл, а лише завершує поточну ітерацію і відразу переходить до наступної.
Коли використовувати: Коли ти хочеш проігнорувати певні елементи (наприклад, порожні рядки або від'ємні числа).
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue; // Пропустити цифру 3 і йти далі до 4
  }
  console.log(i); // Виведе: 1, 2, 4, 5
}

📝 Завдання №10 (Цикл for):
Давай зробимо класичну задачу — виведемо таблицю множення для числа 5, але за допомогою циклу for.
Створи змінну number = 5.
Напиши цикл for, де:
Лічильник i починається з 1.
Цикл працює, поки i менше або дорівнює 10.
На кожному кроці i збільшується на 1.
Всередині циклу виводь шаблонний рядок: ${number} * ${i} = ${number * i}.

✅ Рішення:
let number = 5;

for(let i = 1; i <= 10; i++){
    console.log(`${number} * ${i} = ${number * i}`);
}

📝 Завдання №10.2 (Парні числа):
Давай спробуємо змінити крок циклу.
Напиши цикл for, який виведе в консоль усі парні числа від 2 до 20.
Замість i++, використай крок i += 2.
Виводь просто значення i у кожному кроці.

✅ Рішення:
for(let i = 2; i <= 20; i += 2){
    console.log(i)
}

📝 Завдання №10.3 (Зворотний відлік):
Давай напишемо програму, яка імітує вимкнення пристрою.
Напиши цикл for, який починається з 5 і йде до 1 включно.
На кожному кроці виводь повідомлення: "Система вимкнеться через... ${i}".
Після завершення циклу виведи окремим console.log: "Пристрій вимкнено. 🌑".

✅ Рішення:
for(let i = 5; i >= 1; i--){
    console.log(`Система вимкнеться через... ${i}`)
}
console.log("Пристрій вимкнено. 🌑");

📝 Завдання №10.4 (Аналізатор чисел):
Уяви, що тобі потрібно перевірити список чисел і знайти серед них особливі.
Напиши цикл for, який проходить по числах від 1 до 15 включно.
Всередині циклу для кожного числа зроби перевірку:
Якщо число ділиться на 3 без остачі — виведи: "${i} — ділиться на 3".
В іншому випадку (якщо не ділиться) — виведи просто саме число: "${i}".
Підказка: Для перевірки ділення без остачі використовуй оператор % (залишок від ділення). Якщо i % 3 === 0, то число ділиться на 3.

✅ Рішення:
for(let i = 1; i <= 15; i++){
    if(i % 3 === 0){
        console.log(`${i} — ділиться на 3`);
    } else {
        console.log(`${i}`);
    }
}

📝 Завдання №10.5 (Повний контроль):
Створи масив tasks = ["HTML", "CSS", "JS", "C#", "React", "Python"].
Напиши цикл for, який проходить по цьому масиву.
Використай continue: Якщо навичка — "C#", пропусти її (ми зараз фокусуємося лише на веб-технологіях).
Використай break: Якщо навичка — "React", виведи в консоль: "React ще не вивчав, зупиняюся тут." і повністю зупини цикл.
Для всіх інших навичок виводь у консоль: "Вивчаю: [назва]".

✅ Рішення:
const tasks = ["HTML", "CSS", "JS", "C#", "React", "Python"];

for(let i = 0; i < tasks.length; i ++){
    if(tasks[i] === "C#"){
        continue;
    } else if(tasks[i] === "React"){
        console.log("React ще не вивчав, зупиняюся тут.");
        break;
    } else {
        console.log(`Вивчаю: ${tasks[i]}`);
    }
}
