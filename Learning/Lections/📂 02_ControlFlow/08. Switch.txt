Тема №8: Оператор Switch 🎛️
Оператор switch — це альтернатива довгим ланцюжкам else if. Він порівнює значення однієї змінної з декількома варіантами (кейсами).

1. Основна структура switch 🏗️
Замість того, щоб щоразу писати if (color === "red"), ми кажемо: "Подивись на color і перевір наступні випадки".

2. Ключове слово case 📂
Кожен case — це конкретне значення, з яким ми порівнюємо нашу змінну. Порівняння завжди суворе (===).

3. Навіщо потрібен break? ⛔
Це найважливіша деталь. Якщо JS знайшов потрібний case, він виконає код і піде далі вниз по всіх інших кейсах, 
поки не зустріне break або кінець блоку. break буквально каже: "Досить, ми знайшли те, що шукали, виходь!".

4. Значення за замовчуванням: default 🔄
Це як else у звичайних умовах. Якщо жоден із case не підійшов, виконається код у блоці default.

Приклад коду:
let day = 2;

switch (day) {
  case 1:
    console.log("Понеділок");
    break;
  case 2:
    console.log("Вівторок");
    break;
  default:
    console.log("Інший день");
}

Як ми вже згадували, switch використовує суворе порівняння (===).

📝 Завдання №8 (Switch):
Давай створимо програму для оцінювання результатів тесту.
Створи змінну grade і надай їй значення (наприклад, "B").
Напиши конструкцію switch, яка перевіряє значення grade:
Якщо "A" — виведи: "Відмінно! Ви справжній майстер."
Якщо "B" — виведи: "Добре! Майже ідеально."
Якщо "C" — виведи: "Задовільно. Можна краще."
Якщо "D" — виведи: "Погано. Потрібно більше вчитися."
Для будь-якого іншого значення (наприклад, "F" або випадковий текст) — виведи: "Некоректна оцінка".
Обов'язково постав break після кожного кейсу.

✅ Рішення:
let grade = "B";

switch(grade){
    case "A":
        console.log("Відмінно! Ви справжній майстер.");
        break;
    case "B":
        console.log("Добре! Майже ідеально.");
        break;
    case "C":
        console.log("Задовільно. Можна краще.");
        break;
    case "D":
        console.log("Погано. Потрібно більше вчитися.");
        break;
    default:
        console.log("Некоректна оцінка");
        break;
}

📝 Завдання №8.2 (Групування):
Давай вдосконалимо твою програму оцінювання, щоб вона була менш суворою.
Створи змінну grade (наприклад, "A").
Використай switch для групування:
Якщо "A" або "B" — виведи: "Чудовий результат! Ви пройшли тест."
Якщо "C" або "D" — виведи: "Ви пройшли, але варто повторити матеріал."
Якщо "F" — виведи: "Тест не здано."
default — виведи: "Невідома оцінка".
Спробуй запустити код із значенням "B" і переконайся, що він видає правильну фразу.

✅ Рішення:
let grade = "A";

switch(grade){
    case "A":
    case "B":
        console.log("Чудовий результат! Ви пройшли тест.");
        break;
    case "C":
    case "D":
        console.log("Ви пройшли, але варто повторити матеріал.");
        break;
    case "F":
        console.log("Тест не здано.");
        break;
    default:
        console.log("Невідома оцінка");
        break;
}
