Тема №7: Умовні конструкції (if...else) 🛣️
Уяви, що твій код — це автомобіль на роздоріжжі. Куди він поїде, залежить від умови, яку ми йому дамо.

1. Оператор if (Якщо) 🚦
Це базова перевірка. Код у фігурних дужках {} виконається лише тоді, коли умова в круглих дужках () є істинною (true).

let temp = 25;

if (temp > 20) {
  console.log("На вулиці тепло!");
}

2. Оператор else (Інакше) 🔄
Якщо умова в if виявилася хибною (false), ми можемо дати запасний план.

let age = 15;

if (age >= 18) {
  console.log("Доступ дозволено");
} else {
  console.log("Доступ заборонено (ви занадто молоді)");
}

3. Оператор else if (А якщо...) 🛑
Використовується, коли у нас більше ніж два варіанти розвитку подій.

let score = 75;

if (score >= 90) {
  console.log("Відмінно!");
} else if (score >= 70) {
  console.log("Добре!");
} else {
  console.log("Треба підтягнути знання.");
}

📝 Завдання №7 (Умови):
Давай напишемо програму для перевірки балансу на картці перед покупкою.
Створи дві змінні: balance = 500 та itemPrice = 600.
Напиши конструкцію if...else:
Якщо баланс більший або дорівнює вартості товару, виведи в консоль повідомлення через шаблонний рядок: Покупка успішна! Залишок: [баланс - ціна]$.
Інакше виведи: Недостатньо коштів. Вам бракує [ціна - баланс]$.
Після перевірки спробуй змінити balance на 1000 і подивись, як зміниться результат у консолі.
Уяви, що в магазині діє акція: якщо балансу бракує рівно 100$ або менше, магазин може надати "Кредит до зарплати".
Додай між if та else ще одну умову else if (itemPrice - balance <= 100).
В цьому блоці виведи: Балансу бракує, але ми можемо надати вам кредит!

✅ Рішення:
let balance = 500;
//let balance = 1000;
let itemPrice = 600;

if(balance >= itemPrice){
    let positive = `Покупка успішна! Залишок: ${balance - itemPrice}$`;
    console.log(positive);
} else if (itemPrice - balance <= 100){
    console.log("Балансу бракує, але ми можемо надати вам кредит!");
} else {
    let negative = `Недостатньо коштів. Вам бракує ${itemPrice - balance}$`;
    console.log(negative);
}

📝 Завдання №7.2 (Світлофор):
Давай спробуємо реалізувати логіку світлофора за допомогою if / else if / else.
Створи змінну signal = "yellow".
Напиши умови:
Якщо signal === "green", виведи: "Йди!".
Якщо signal === "yellow", виведи: "Приготуйся!".
Якщо signal === "red", виведи: "Стій!".
В усіх інших випадках (наприклад, якщо хтось ввів "blue") виведи: "Світлофор зламався".
Не забудь про крапки з комою ; після кожної команди console.log() та створення змінних.

✅ Рішення:
let signal = "yellow";

if(signal === "green"){
    console.log("Йди!");
} else if(signal === "yellow"){
    console.log("Приготуйся!");
} else if(signal === "red"){
    console.log("Стій!");
} else {
    console.log("Світлофор зламався");
}
