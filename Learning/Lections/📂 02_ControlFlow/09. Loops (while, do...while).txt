Тема №9: Цикли (Loops) — while та do...while 🔄
Цикли дозволяють виконувати один і той самий блок коду багато разів, поки справджується певна умова.

1. Цикл while (Поки) ⏳
Він спочатку перевіряє умову, і якщо вона істинна — виконує код. Це схоже на if, який повторюється знову і знову.
⚠️ Важливо: Всередині циклу обов'язково має бути щось, що змінить умову (наприклад, збільшення числа), 
інакше цикл стане нескінченним і "повісить" браузер або комп'ютер.
let count = 1;

while (count <= 3) {
  console.log(`Це коло №${count}`);
  count++; // Збільшуємо на 1, щоб колись вийти з циклу
}

2. Інкремент (++) та Декремент (--) ➕➖
Це короткий запис математики, який ти часто бачитимеш у циклах:
count++ — це те саме, що count = count + 1.
count-- — це те саме, що count = count - 1.

3. Цикл do...while (Зробити... поки) 🔁
Його особливість у тому, що він спочатку виконує код, а вже потім перевіряє умову. 
Це гарантує, що код виконається хоча б один раз, навіть якщо умова з самого початку була хибною.
let energy = 0;

do {
  console.log("Я роблю один крок...");
} while (energy > 0); 
// Виведе текст один раз, хоча енергія вже на нулі

📝 Завдання №9 (Цикли):
Давай створимо зворотний відлік для запуску ракети!
Створи змінну counter і надай їй значення 5.
Використай цикл while, щоб виводити в консоль текст: "До запуску залишилось: [значення counter]".
Всередині циклу зменшуй counter на одиницю при кожному кроці.
Цикл має працювати, поки counter більший за 0.
Після завершення циклу (поза його дужками) виведи в консоль: "Поїхали! 🚀".

✅ Рішення:
let counter = 5;
while(counter > 0){
    console.log(`До запуску залишилось: ${counter}`);
    counter--;
}
console.log("Поїхали! 🚀");

📝 Завдання №9.2 (Спроба входу):
Уяви, що ми даємо користувачу можливість ввести пароль, але в нашому навчальному прикладі ми просто імітуємо одну спробу.
Створи змінну attempts = 0.
Використай конструкцію do...while.
Всередині блоку do:
Виведи в консоль: "Спроба №[attempts + 1]. Введіть пароль...".
Збільш attempts на 1 (attempts++).
В умові while напиши, щоб цикл тривав, поки attempts < 0 (це завідомо хибна умова, бо ми вже зробили спробу і attempts став рівним 1).
Подивись, чи виведеться текст у консоль.

✅ Рішення:
let attempts = 0;

do {
    console.log(`Спроба №${attempts + 1}. Введіть пароль...`);
    attempts ++;
} while (attempts < 0);

📝 Завдання №9.3 (Генератор секретного коду):
Уяви, що твоя програма має згенерувати "код підтвердження", але через технічні особливості 
(наприклад, слабкий сигнал) вона може робити це кілька разів, поки не отримає потрібне значення.
Створи змінну secretKey = 0.
Використай конструкцію do...while.
Всередині блоку do:
Додавай до secretKey число 15 (secretKey = secretKey + 15).
Виводь у консоль повідомлення: "Генерація... Поточне значення ключа: [secretKey]".
В умові while вкажи, щоб цикл продовжувався, поки secretKey < 100.
Після завершення циклу виведи: "Ключ згенеровано успішно! Підсумкове значення: [secretKey]".

✅ Рішення:
let secretKey = 0;

do {
    secretKey += 15;
    console.log(`Генерація... Поточне значення ключа: ${secretKey}`);
} while (secretKey < 100);
console.log(`Ключ згенеровано успішно! Підсумкове значення: ${secretKey}`)
