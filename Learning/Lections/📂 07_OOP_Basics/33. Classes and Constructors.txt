Тема №33: Вступ до ООП. Класи та конструктори (Introduction to OOP. Classes and Constructors) 🏗️
Об'єктно-орієнтоване програмування (ООП) — це підхід до написання коду, де ми моделюємо реальні об'єкти з нашого світу у вигляді програмних сутностей. 
Замість того, щоб просто писати окремі функції, ми створюємо "креслення" для об'єктів.

🔵 Класи (Classes): Це шаблони або схеми, за якими створюються об'єкти. 
Клас визначає, які дані (властивості) та які дії (методи) будуть мати всі об'єкти цього типу.

⚙️ Конструктор (Constructor): Це спеціальний метод всередині класу, який автоматично запускається в момент створення нового об'єкта (екземпляра). 
Він дозволяє нам встановити початкові значення для властивостей об'єкта.

📦 Екземпляр (Instance): Це конкретний об'єкт, створений на основі класу. 
У нас може бути один клас "Автомобіль", але сотні екземплярів цього класу, кожен з яких матиме свій унікальний колір, марку чи швидкість.

🔑 Ключове слово this: У контексті класів this посилається саме на той конкретний екземпляр, який ми створюємо прямо зараз. 
Це дозволяє нам сказати: "запиши це ім'я цьому об'єкту".

📝 Синтаксис створення:
class НазваКласу {
  constructor(параметр1, параметр2) {
    this.властивість1 = параметр1;
    this.властивість2 = параметр2;
  }
}

const мійОб'єкт = new НазваКласу('значення1', 'значення2');

Тепер, коли у нас є база, давай закріпимо це на простому прикладі. Як ми домовлялися, почнемо з елементарного.

📝 Завдання №33.1 (Створення класу "Користувач"):
Створи клас під назвою User. У конструкторі цього класу мають бути два параметри: name (ім'я) та age (вік). 
Всередині конструктора привласни ці параметри властивостям this.name та this.age.
Після створення класу, створи новий екземпляр (об'єкт) з іменем 'Іван' та віком 25 років, і запиши його у змінну user1. 
Виведи user1 у консоль, щоб переконатися, що об'єкт створено правильно.

✅ Рішення:
class User {
    constructor(name, age){
        this.name = name;
        this.age = age;
    }
}

const user1 = new User('Ivan', 25);
console.log(user1);

📝 Завдання №33.2 (Клас "Car"):
Створи клас Car.
У конструкторі зроби параметри для brand (марка), model (модель) та year (рік випуску).
Привласни їх властивостям this.brand, this.model, this.year.
Створи два різні об'єкти (два різні екземпляри) цього класу: наприклад, car1 (твоя улюблена машина) та car2 (будь-яка інша).
Виведи обидва об'єкти в консоль.

✅ Рішення:
class Car {
    constructor(brand, model, year){
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
}

const volkswahen = new Car('Volkswagen', 'Golf', 2013);
const lamborghini = new Car ('Lamborgini', 'Miura', 1968);
console.log (volkswahen);
console.log(lamborghini);
