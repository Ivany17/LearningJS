Тема №18: Деструктуризація масивів (Array Destructuring) 🪜
Якщо в об'єктах ми діставали дані за іменами ключів ({ name }), то в масивах все базується на порядку (індексах). 
Ми використовуємо квадратні дужки [] замість фігурних {}.
Деструктуризація масивів дозволяє нам «розпаковувати» значення з масиву у окремі змінні одним рядком коду. 
У C# для цього зазвичай доводиться звертатися до кожного елемента за індексом окремо, а в JS ми робимо це масово.

1. Основний синтаксис 📦
Ми використовуємо квадратні дужки [] зліва від знака рівності. Назви змінним ми даємо самі, але їхнє значення залежить від порядку в масиві.
const fruits = ["🍎", "🍌", "🥝"];
// Деструктуризація
const [apple, banana, kiwi] = fruits;
console.log(apple); // "🍎"

2. Пропуск елементів ⏭️
Якщо нам не потрібні всі значення, ми можемо просто залишити порожнє місце між комами.
const [first, , third] = ["A", "B", "C"];
// first = "A", third = "C" (другий елемент пропущено)

3. Значення за замовчуванням 🛡️
Як і в об'єктах, якщо масив коротший за наш список змінних, ми можемо призначити «запасне» значення.
const [name = "Гість"] = [];

4. Оператор rest ... (решта) 🎒
Ми можемо зібрати всі елементи, що залишилися, у новий масив.
const [leader, ...team] = ["Ivan", "Oleg", "Anna", "Dmitry"];
// leader = "Ivan", team = ["Oleg", "Anna", "Dmitry"]

5. Обмін значеннями (Swapping variables) 🔄
У JavaScript деструктуризація дозволяє міняти значення двох змінних місцями 
без створення додаткової тимчасової змінної (як це часто роблять у C#).
Ось як це виглядає:
let a = 1;
let b = 2;

[a, b] = [b, a]; // Тепер a = 2, b = 1

📝 Завдання №18: Призери забігу 🏃‍♂️
Уяви, що в нас є результати марафону. Перші три учасники отримують золото, срібло та бронзу відповідно, а всі інші отримують дипломи учасників.
Створи масив raceResults: ["Ivan", "Oleg", "Anna", "Dmytro", "Maria", "Ihor"].
Використай деструктуризацію, щоб:
Створити змінні gold, silver та bronze для перших трьох місць.
Створити масив others, куди потраплять усі інші учасники (використай оператор ...).
Виведи в консоль результати:
"Переможці: [gold], [silver], [bronze]."
"Інші учасники: [others]."

✅ Рішення:
const raceResults = ["Ivan", "Oleg", "Anna", "Dmytro", "Maria", "Ihor"];
const [gold, silver, bronze, ...others] = raceResults;
console.log(`Переможці: ${gold}, ${silver}, ${bronze}.`);
console.log(`Інші учасники: ${[others]}.`);

📝 Завдання №18.2: Командна ротація (Swapping variables)
Уяви, що в грі два гравці вирішили помінятися своїми ролями.
Створи дві змінні: let player1 = "Warrior"; та let player2 = "Mage";.
Використай деструктуризацію масивів, щоб поміняти їхні значення місцями одним рядком коду.
Виведи результат у консоль одним рядком: "Гравець 1 тепер: [player1], а Гравець 2: [player2]".

✅ Рішення:
let player1 = "Warrior";
let player2 = "Mage";

[player1, player2] = [player2, player1];

console.log(`Гравець 1 тепер: ${player1}, а Гравець 2: ${player2}`);

📝 Завдання №18.3: Робота з вкладеними масивами
Уяви, що в тебе є дані про координати двох гравців на карті:
Створи змінну const playersData = [["Archer", 10, 25], ["Mage", 40, 50]];
Використай деструктуризацію, щоб одним рядком дістати дані тільки для першого гравця у змінні: playerName, x, y.
Підказка: тобі потрібно деструктуризувати перший елемент великого масиву, який сам є масивом: const [[playerName, x, y]] = ...
Виведи в консоль повідомлення: "Гравець [playerName] знаходиться в координатах [x]:[y]".

✅ Рішення:
const playersData = [["Archer", 10, 25], ["Mage", 40, 50]];
const [[playerName1, x1, y1], [playerName2, x2, y2]] = playersData;
console.log(`Гравець ${playerName1} знаходиться в координатах ${x1}:${y1}`);
console.log(`Гравець ${playerName2} знаходиться в координатах ${x2}:${y2}`);
