Тема №16: Методи об'єктів та слово this (Функції всередині) ⚙️
Об'єкти можуть містити не лише дані (рядки, числа, масиви), а й дії. Коли функція є частиною об'єкта, вона називається методом.

1. Як створити метод?
Метод — це функція, яка є властивістю об'єкта. Якщо властивість описує характеристику (який предмет?), то метод описує дію (що він робить?).

2. Ключове слово this 🔑
Коли ти всередині метода хочеш звернутися до імені героя або його рівня, ти не можеш просто написати nickname. 
Тобі потрібно сказати: "візьми ім'я з цього об'єкта". Для цього пишемо this.nickname.

Приклад:
const cat = {
  name: "Мурчик",
  // Це метод:
  sayHi: function() {
    console.log(`Мяу! Мене звати ${this.name}`); // this.name звертається до "Мурчик"
  }
};

cat.sayHi(); // Виклик методу (обов'язково з дужками!)

3. Скорочений запис методів
У сучасному JS (ES6+) метод можна записувати простіше, без слова function та двокрапки. Це робить код чистішим.
const warrior = {
  name: "Warrior",
  // Замість fight: function() { ... }
  fight() {
    this.stamina -= 10;
    console.log(`Залишилося ${this.stamina} витривалості`);
  }
};

4. Проблема стрілочних функцій => в об'єктах
Це критично важливо: ніколи не використовуй стрілочні функції для методів об'єктів, якщо тобі потрібен this.
Стрілочні функції не мають свого this. Вони беруть його "ззовні" (з глобального оточення). 
Якщо ти напишеш метод як () => { console.log(this.name) }, то отримаєш undefined.

5. Аргументи в методах
Метод — це та ж сама функція, тому ми можемо передавати в нього значення.

6. Приклад із життя:
Уяви, що воїн знаходить аптечку. Ми не знаємо заздалегідь, наскільки вона сильна.
const warrior = {
  health: 50,
  heal(amount) {
    this.health += amount;
    console.log(`Поповнено здоров'я на ${amount}. Тепер: ${this.health}`);
  }
};

warrior.heal(30); // Передаємо 30 як аргумент

7. У реальних проектах об'єкти рідко існують поодинці. Найчастіше вони зібрані в масиви. 
Це дозволяє нам керувати цілими групами (користувачами, товарами в кошику або героями в загоні).
Це масив, де кожним елементом є об'єкт.
const team = [
  { name: "Warrior", health: 100 },
  { name: "Mage", health: 70 },
  { name: "Healer", health: 80 }
];

8. Як з цим працювати?
Ти можеш поєднувати знання про масиви та об'єкти. Наприклад, використати .forEach(), щоб кожен герой у команді щось зробив.
team.forEach(hero => {
  console.log(`Герой: ${hero.name}`);
});

📝 Завдання №16.1: (Оживлення героя):
Давай створимо об'єкт, який не просто зберігає інформацію, а вміє нею користуватися.
Створи об'єкт warrior.
Додай йому властивості:
name (будь-яке ім'я).
health (число, наприклад 100).
stamina (витривалість, число 50).
Додай метод showStats: він має виводити повідомлення: "Воїн [name] має [health] HP та [stamina] витривалості". (Використовуй this).
Додай метод fight: він має зменшувати stamina на 10 одиниць і виводити: "Атака! Залишилося [stamina] витривалості".
Виклич showStats(), потім один раз fight(), і знову showStats(), щоб побачити, як змінилися дані.

✅ Рішення:
const warrior = {
    name: "Warrior",
    health: 89,
    stamina: 46,
    showStats: function(){
        console.log(`Воїн ${this.name} має ${this.health} HP та ${this.stamina} витривалості`);
    },
    fight: function(){
        this.stamina -= 10;
        console.log(`Атака! Залишилося ${this.stamina} витривалості`);
    },
}

warrior.showStats();
warrior.fight();
warrior.showStats();

Додай новий метод rest (відпочинок):
Він має додавати +20 до stamina.
Але є умова: stamina не може стати більше за 100.
Використай if/else, щоб перевірити: якщо після додавання стає більше 100, прирівняй її просто до 100.

const warrior = {
    name: "Warrior",
    health: 89,
    stamina: 46,
    showStats(){
        console.log(`Воїн ${this.name} має ${this.health} HP та ${this.stamina} витривалості`);

    }
}

warrior.rest(); // Warrior відпочив. Тепер витривалість: 56
warrior.rest(); // Warrior відпочив. Тепер витривалість: 76
warrior.rest(); // Warrior відпочив. Тепер витривалість: 96
warrior.rest(); // Warrior відпочив. Тепер витривалість: 100
warrior.rest(); // Warrior відпочив. Тепер витривалість: 100

📝 Завдання №16.3 (Тренування та Шкода):
Давай додамо динаміки твоєму воїну.
Використай свій об'єкт warrior.
Додай метод takeDamage(damage):
Він має приймати кількість шкоди як параметр.
Віднімати цю шкоду від this.health.
Виводити: "Воїн [name] отримав [damage] шкоди. Залишилося [health] HP".
Додай метод training(hours):
За кожну годину тренування (hours) воїн отримує +5 до витривалості (stamina).
Метод має приймати кількість годин, множити їх на 5 і додавати до stamina.
Виведи: "Після [hours] годин тренувань витривалість зросла до [stamina]".
Виклич takeDamage(25), а потім training(3).

✅ Рішення:
const warrior = {
    name: "Warrior",
    health: 89,
    stamina: 46,
    showStats(){
        console.log(`Воїн ${this.name} має ${this.health} HP та ${this.stamina} витривалості`);
    },
    fight(){
        this.stamina -= 10;
        console.log(`Атака! Залишилося ${this.stamina} витривалості`);
    },
    rest(){
        this.stamina += 20;
        if(this.stamina > 100){
            this.stamina = 100;
        }
        console.log(`${this.name} відпочив. Тепер витривалість: ${this.stamina}`);
    },
    takeDamage(damage){
        this.health -= damage;
        console.log(`Воїн ${this.name} отримав ${damage} шкоди. Залишилося ${this.health} HP`);
    },
    training(hours){
        this.stamina += hours * 5;
        console.log(`Після ${hours} годин тренувань витривалість зросла до ${this.stamina}`)
    },
}

warrior.takeDamage(25);
warrior.training(3);
warrior.showStats();

Створи масив squad, який містить 3 об'єкти. Кожен об'єкт — це воїн із властивостями name та stamina.
Використай метод .forEach(), щоб пройтися по кожному воїну в масиві.
Дія всередині .forEach():
Додай кожному воїну +15 до витривалості (stamina).

    { name: "Iron Man", stamina: 83 },
    { name: "Spiderman", stamina: 75 },
