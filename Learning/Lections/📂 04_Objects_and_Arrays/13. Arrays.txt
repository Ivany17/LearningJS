Тема №13: Масиви (Arrays) 📚
Масив — це спеціальний тип даних, який дозволяє зберігати впорядковані колекції значень під одним ім'ям.
У JavaScript масиви можуть містити будь-які типи даних одночасно (числа, рядки, булеві значення і навіть інші масиви).

1. Навіщо вони потрібні?
Уяви, що ти розробляєш список покупок. Без масивів тобі довелося б створювати змінні item1, item2, item3... 
Це незручно. Масив дозволяє покласти все в одну "коробку" з секціями.

2. Створення масиву
Масиви записуються у квадратних дужках [], а елементи розділяються комами.
const mySkills = ["Console", "Variables", "Functions"];
console.log(mySkills[0]); // "Console"
mySkills[1] = "New Value"; // Зміна елемента за індексом

3. Індексація (Як дістати елемент?)
⚠️ Це найважливіший момент: відлік у масивах починається з 0.
Перший елемент має індекс 0.
Другий — індекс 1.
І так далі.

4. Властивість .length
Кожен масив автоматично знає, скільки в ньому елементів. Це можна дізнатися за допомогою команди .length.
console.log(mySkills.length); // Виведе 3

5. Корисні властивості та методи пошуку
.indexOf(item) — шукає елемент і повертає його індекс. Якщо не знайдено — повертає -1.
.includes(item) — перевіряє, чи є такий елемент у масиві. Повертає true або false.
.push(item) - додає елемент в кінець 
mySkills.push("Arrays");

.unshift(item) - додає елемент на початок 
mySkills.unshift("Intro");

.pop() - видаляє останній елемент 
mySkills.pop();

.shift() - видаляє перший елемент 
mySkills.shift();

📝 Завдання №13.3 (Цикл + Масив):
Давай автоматизуємо вивід твоїх навичок.
Використай свій масив mySkills.
Напиши цикл for, який проходить від 0 до mySkills.length.
Всередині циклу виведи кожен елемент у форматі: "Я вже знаю: [назва навички]".
Додаткова зірочка ⭐: Додай умову if всередині циклу. Якщо навичка — "Functions", виведи після назви додатково: "(Це було складно, але я зміг!)".

✅ Рішення:

for(let i = 0; i < mySkills.length; i++){
    if(mySkills[i] === "Functions"){
        console.log(`Я вже знаю: ${mySkills[i]} (Це було складно, але я зміг!)`);
    } else {
        console.log(`Я вже знаю: ${mySkills[i]}`);
    }
}

📝 Завдання №13.4 (Складний пошук):
Уяви, що ти робиш систему управління замовленнями.
Створи масив orders. Поклади туди 5 чисел (це суми замовлень), наприклад: [500, 150, 1200, 300, 800].
Напиши функцію (можна звичайну або стрілочну), яка називається findBigOrders. Вона повинна:
Приймати масив як параметр.
Створити всередині новий порожній масив, наприклад let bigOnes = [];.
За допомогою циклу for пройтися по вхідному масиву.
Якщо сума замовлення більше 400, "заштовхнути" її в новий масив bigOnes (пам'ятаєш про метод .push()?).
Повернути (return) цей новий масив з великими сумами.
Виклич цю функцію, передавши їй свій масив orders, і збережи результат у нову змінну.
Виведи результат у консоль.

✅ Рішення:
const orders = [500, 150, 1200, 300, 800];

const findBigOrders = orders => {
    let bigOnes = [];
    for(let i = 0; i < orders.length; i++){
        if(orders[i] > 400){
            bigOnes.push(orders[i]);
        }
    }
    return bigOnes;
};

const newOrders = findBigOrders(orders);
console.log(newOrders);
