Тема №15: Об'єкти (Objects) 💎
До цього ми працювали з масивами — це просто списки. Але в реальному програмуванні нам потрібно описувати складні речі.

1. Що таке об'єкт? 🤔
Уяви, що ти описуєш користувача. В масиві це було б: ["Іван", 25, "Київ"]. Але як зрозуміти, де тут ім'я, а де місто?
Об'єкт дозволяє дати кожному значенню зрозуміле ім'я (ключ).

2. Синтаксис ✍️
Об'єкт створюється за допомогою фігурних дужок {}. Всередині ми пишемо пари ключ: значення.
const user = {
  name: "Іван",
  age: 25,
  city: "Київ"
};

3. Як працювати з даними? 🛠️
Найпростіший спосіб дістати або змінити значення — це використати крапку:
console.log(user.name); // Виведе: Іван
user.age = 26; // Змінить вік на 26

Об'єкти в JavaScript дуже гнучкі. Ти можеш додавати нові характеристики предмета або видаляти старі вже після того, як об'єкт був створений.
4. Додавання нової властивості
Щоб додати щось нове, просто вигадай назву ключа і присвой йому значення через крапку.
const car = { brand: "Tesla" };
car.color = "Red"; // Тепер в об'єкті є колір

5. Видалення властивості
Для цього існує спеціальний оператор delete.
delete car.color; // Колір зникне з об'єкта

📝 Завдання №15.1: (Твій перший об'єкт):
Спробуй описати свій улюблений смартфон (або будь-який інший предмет) як об'єкт.
Створи об'єкт phone.
Додай йому мінімум 4 властивості:
brand (рядок, наприклад "Apple" або "Samsung").
model (рядок).
price (число).
isWorking (булеве значення true або false).
Виведи в консоль повідомлення, використовуючи шаблонний рядок: "Мій телефон — це [brand] [model], і він коштує [price] грн."
Зміни ціну (price) в об'єкті на будь-яку іншу і виведи об'єкт повністю в консоль (console.log(phone)), щоб побачити зміни.

✅ Рішення:
const phone = {
    brand: "Samsung",
    model: "Galaxy A26",
    price: 11699,
    isWorking: true,
}

console.log(`Мій телефон — це ${phone.brand} ${phone.model}, і він коштує ${phone.price} грн.`)
phone.price = 15000;
console.log(phone);

📝 Завдання №15.2 (Модернізація об'єкта):
Давай продовжимо працювати з твоїм об'єктом phone.
Візьми свій попередній код з об'єктом phone.
Додай йому нову властивість owner (рядок з твоїм ім'ям).
Додай властивість apps (масив із 3-х назв додатків, наприклад ["Instagram", "TikTok", "YouTube"]).
Видали властивість isWorking.
Виведи об'єкт у консоль, щоб переконатися, що:
З'явився власник та масив додатків.
Зник статус isWorking.
Зірочка (на логіку): Спробуй вивести в консоль лише другий додаток із масиву, який лежить всередині об'єкта.

✅ Рішення:
const phone = {
    brand: "Samsung",
    model: "Galaxy A26",
    price: 11699,
    isWorking: true,
}

phone.owner = "Іван";
phone.apps = ["Instagram", "TikTok", "YouTube"];
delete phone.isWorking;

console.log(phone);
console.log(phone.apps[1]);

📝 Завдання №15.3 (Профіль героя):
Створи об'єкт hero.
Додай властивості:
nickname (рядок).
level (число).
isOnline (булеве).
equipment (масив із 3-х предметів, наприклад: ["Меч", "Щит", "Шолом"]).
Виконай маніпуляції:
Збільш level на одиницю (через hero.level = ...).
Додай нову властивість guild (назва гільдії).
Видали властивість isOnline (герой ніби вийшов із гри).
Виведи в консоль: "Герой [nickname] має у спорядженні: [другий предмет із масиву equipment]".
Виведи весь об'єкт у консоль.

✅ Рішення:
const hero = {
    nickname: "Aragorn",
    level: 14,
    isOnline: false,
    equipment: [ "Меч", "Щит", "Шолом" ]
}

hero.level += 1;
hero.guild = "Archer";
delete hero.isOnline;

console.log(`Герой ${hero.nickname} має у спорядженні: ${hero.equipment[1]}`);
console.log(hero);
