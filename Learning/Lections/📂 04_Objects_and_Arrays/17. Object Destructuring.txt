Тема №17: Деструктуризація об'єктів (Object Destructuring) 🪄
Це "магічний" спосіб швидко діставати дані з об'єкта та записувати їх у змінні. Це робить код набагато чистішим.

1. Навіщо це треба?
Уяви, що у тебе є об'єкт користувача:
const user = { name: "Іван", age: 25, city: "Київ" };

Замість того, щоб писати:
const name = user.name;
const age = user.age;

Ти можеш зробити це в один рядок:
const { name, age } = user; // JS сам знайде ключі 'name' та 'age' і створить такі змінні

2. Перейменування
Якщо тобі потрібно, щоб змінна називалася інакше, ніж ключ в об'єкті:
const { name: userName } = user; // Тепер у тебе є змінна userName зі значенням "Іван"

📝 Завдання №17.1 (Швидкий доступ):
Давай спростимо роботу з твоїм воїном.
Створи об'єкт boss із властивостями: title: "Dragon", hp: 5000, location: "Cave".
Використай деструктуризацію, щоб в один рядок витягнути title та location у змінні.
Виведи в консоль повідомлення: "У локації [location] з'явився бос — [title]!".
Спробуй ще раз деструктуризувати hp, але перейменуй цю змінну на bossHealth. Виведи її в консоль.

✅ Рішення:
const boss = {
    title: "Dragon",
    hp: 5000,
    location: "Cave",
}

const { title, location } = boss;
console.log(`У локації ${location} з'явився бос — ${title}!`);
const { hp: health } = boss; // перейменовуємо hp на health
console.log(health);

📝 Завдання №17.2: Глибока деструктуризація
Уяви, що ти отримуєш дані про замовлення в інтернет-магазині.
Створи об'єкт order, де будуть id, product і вкладений об'єкт details, який містить color і warranty.
Твоє завдання — одним рядком деструктуризувати цей об'єкт так, щоб отримати три змінні: product, color та warranty.
Підказка: Синтаксис для вкладених об'єктів виглядає так: const { details: { color } } = order;
Виведи в консоль повідомлення: "Товар: [product], Колір: [color], Гарантія: [warranty]".
Додатково (на перейменування): Деструктуризуй id, але збережи його у змінну з назвою orderId. Виведи її в консоль.

✅ Рішення:
const order = {
    id: 5542,
    product: "Клавіатура",
    details: {
        color: "Black",
        warranty: "12 місяців"
    }
};

const { product, details: { color, warranty } } = order;
console.log(`Товар: ${product}, Колір: ${color}, Гарантія: ${warranty}`);
const { id: orderId } = order;
console.log(orderId);

📝 Завдання №17.3: Повна автономія (Профіль Студента)
Уяви, що ти створюєш систему обліку для курсів програмування.
Створи об'єкт student, який має:
Ім'я (firstName).
Прізвище (lastName).
Вкладений об'єкт progress, у якому є ключі topic (назва теми, наприклад "JavaScript") та score (число).
Виконай деструктуризацію в один рядок, щоб отримати:
Змінну name (сюди має потрапити значення з firstName).
Змінну topic (з вкладеного об'єкта).
Змінну points (сюди має потрапити значення зі score).
Виведи в консоль результат у такому вигляді:
"Студент [name] вивчає [topic] і має [points] балів."
Спробуй змінити значення score в оригінальному об'єкті та виклич деструктуризацію ще раз (або просто виведи оновлений бал), 
щоб переконатися, що ти розумієш, як дані синхронізуються (або не синхронізуються) після створення змінних.

✅ Рішення:
const student = {
    firstName: "Peter",
    lastName: "Parker",
    progress: {
        topic: "JavaScript",
        score: 86,
    },
}

const { firstName: name, progress: { topic, score: points } } = student;
console.log(`Студент ${name} вивчає ${topic} і має ${points} балів.`);
