Тема №26: Метод .sort() 📊
Цей метод використовується для сортування елементів масиву.

⚠️ ВАЖЛИВА ВІДМІННІСТЬ ВІД C#: У LINQ .OrderBy() створює нову послідовність, не змінюючи оригінал. 
В JavaScript метод .sort() мутує (змінює) оригінальний масив.

🔹 Як він працює за замовчуванням?
Без параметрів .sort() перетворює всі елементи на рядки і порівнює їх за таблицею Unicode. 
Це призводить до дивних результатів із числами: наприклад, число 10 стане перед 2, бо символ "1" йде раніше за "2".

🔹 Функція порівняння (Compare Function)
Щоб сортувати числа або об'єкти правильно, ми передаємо в .sort((a, b) => ...) спеціальну функцію:
Якщо результат a - b від'ємний — a ставиться перед b.
Якщо додатний — b ставиться перед a.
Якщо 0 — порядок не змінюється.

Приклад для чисел:
const nums = [10, 2, 5];
nums.sort((a, b) => a - b); // [2, 5, 10] — за зростанням
nums.sort((a, b) => b - a); // [10, 5, 2] — за спаданням

✏️ Сортування рядків (String Sorting) 🔤
Для правильного сортування тексту (за алфавітом) використовується метод рядків .localeCompare(). 
Він порівнює два рядки з урахуванням мовних особливостей (враховує регістр, специфічні символи тощо).

🔹 Як це працює:
stringA.localeCompare(stringB) повертає:
-1 (або інше від'ємне число), якщо stringA має йти перед stringB.
1 (або додатне число), якщо stringB має йти перед stringA.
0, якщо вони однакові.

Це ідеально підходить для нашої функції порівняння в .sort().

Приклад:
const names = ["Ivan", "Anna", "Oleg"];
names.sort((a, b) => a.localeCompare(b)); // ["Anna", "Ivan", "Oleg"]
