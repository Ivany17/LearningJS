Тема №20: Методи масивів. Метод .map() 🗺️
Це один із найважливіших методів у JavaScript. Якщо .forEach() просто виконує дію для кожного елемента, то .map() 
створює новий масив на основі старого, трансформуючи кожен його елемент.

Головні правила .map():
Незмінність (Immutability): Оригінальний масив залишається таким, яким він був. Це дуже важливо в сучасній розробці.
Повернення (Return): Функція всередині .map() обов'язково має щось повертати, інакше в новому масиві будуть undefined.

1. Чим він відрізняється від .forEach()?
.forEach() — нічого не повертає. Він просто "пробігає" по масиву (наприклад, щоб щось вивести в консоль).
.map() — повертає новий масив такої ж довжини, де кожен елемент змінений за твоїм правилом.

2. Приклад:
const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2); 
// doubled тепер [2, 4, 6]

📝 Завдання №20.1 (Трансформація даних):
Уяви, що ти розробник і отримуєш список цін у доларах, але тобі потрібно відобразити їх у гривнях.
Створи масив pricesUSD із п'ятьма числами (наприклад, [10, 25, 50, 100, 5]).
Створи новий масив pricesUAH за допомогою методу .map().
Логіка трансформації: помнож кожну ціну на актуальний курс (наприклад, 40).
Виведи обидва масиви в консоль, щоб побачити різницю.

✅ Рішення:
const pricesUSD = [10, 25, 50, 100, 5];
const pricesUAH = pricesUSD.map(num => num * 40);
console.log(pricesUSD);
console.log(pricesUAH);

📝 Завдання №20.2: Створення об'єктів через .map()
У нас є масив назв предметів, які гравець знайшов у скрині. Нам потрібно перетворити ці назви на повноцінні об'єкти з додатковими властивостями.
Створи масив items із назвами: ["Sword", "Shield", "Potion"].
Створи масив gameInventory за допомогою методу .map().
Логіка трансформації: для кожного рядка (назви предмета) поверни об'єкт, який має такі ключі:
name: сама назва предмета.
isNew: значення true.
id: випадкове число (можеш використати Math.random()).
Виведи gameInventory в консоль.

✅ Рішення:
const items = ["Sword", "Shield", "Potion"];
const gameInventory = items.map(item => ({ name : item, isNew: true, id : Math.floor(Math.random() * 100 + 1) }));
console.log(gameInventory);

📝 Завдання №20.3: Створення HTML-розмітки
Уяви, що ти отримуєш імена користувачів і тобі потрібно перетворити їх на список для відображення на сайті.
Створи масив users: ["Ivan", "Oleg", "Anna"].
Використай .map(), щоб створити новий масив userListItems.
Логіка трансформації: Кожне ім'я має перетворитися на рядок з HTML-тегом: <li>Ім'я</li> (тут тобі знадобляться шаблонні рядки).
Виведи userListItems у консоль.

✅ Рішення:
const users = ["Ivan", "Oleg", "Anna"];
const userListItems = users.map(name => `<li> ${name} </li>`);
console.log(userListItems);

📝 Завдання №20.4: Створення об'єктів з ID
Створи масив ids: [101, 102, 103].
Використай .map(), щоб створити новий масив userObjects.
Логіка трансформації: Для кожного числа (id) поверни об'єкт, який має дві властивості:
userId: саме це число (id).
role: рядок "guest".
Порада: Якщо ти пишеш стрілочну функцію в один рядок і повертаєш об'єкт, не забудь взяти тіло функції в круглі дужки: 
id => ({ userId: id, role: "guest" }). Без дужок JS сприйме фігурні дужки як початок блоку коду, а не як об'єкт.
Виведи userObjects у консоль.

✅ Рішення:
const ids = [101, 102, 103];
const userObjects = ids.map(id => ({userId: id, role: "guest"}));
console.log(userObjects);

📝 Завдання №20.5: Фінальна обробка даних 📊
Уяви, що ти розробник в інтернет-магазині. Тобі приходить масив "сирих" даних про товари, і тобі потрібно підготувати їх для відображення на сайті.
Початкові дані: Створи масив rawProducts, де у кожного продукта буде свій id, name ("Sofa", "Chair", "Table" і priceUSD (250, 75, 120)).
Завдання: Створи новий масив formattedProducts за допомогою .map().
Логіка трансформації: для кожного об'єкта в масиві:
Використай деструктуризацію, щоб дістати name та priceUSD.
Поверни новий об'єкт, який містить:
Властивість title (це те саме, що було в name, але перейменоване).
Властивість priceUAH (ціна в доларах, помножена на 40).
Властивість category зі значенням "Furniture".
Властивість isCheap, яка має бути true, якщо ціна в гривнях менша за 4000, і false, якщо більша або рівна.
Результат: Виведи formattedProducts у консоль.

✅ Рішення:
const rawProducts = [
    { id: 1, name: "Sofa", priceUSD: 250 },
    { id: 2, name: "Chair", priceUSD: 75 },
    { id: 3, name: "Table", priceUSD: 120 }
];


const formattedProducts = rawProducts.map(product => {
        const { name, priceUSD } = product;
        const priceUAH = priceUSD * 40;
        return { title : name, 
                priceUAH : priceUAH, 
                category : "Furniture", 
                isCheap : priceUAH >= 4000 ? false : true };
    }
);

console.log(formattedProducts);
