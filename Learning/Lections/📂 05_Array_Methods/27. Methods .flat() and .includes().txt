Тема №27: Методи .flat() та .includes() 🗂️

🔹 Метод .flat()
Цей метод використовується для "розгладжування" масиву. Якщо у вас є масив, який містить у собі інші масиви (вкладеність), 
.flat() витягне елементи з внутрішніх масивів і зробить один плоский масив.
За замовчуванням він розгладжує лише на 1 рівень глибини.
Аналог у C# LINQ: .SelectMany().

🔹 Метод .includes()
Це дуже простий метод-перевірка. Він повертає true, якщо масив містить певне значення, і false, якщо ні. 
Він працює за принципом строгої рівності (===).
Аналог у C# LINQ: .Contains().

Приклад:
const nested = [1, [2, 3], 4];
nested.flat(); // [1, 2, 3, 4]

const fruits = ['apple', 'banana'];
fruits.includes('apple'); // true

🤿 Метод .flat(depth) приймає один необов'язковий аргумент — depth (глибина).
За замовчуванням depth = 1.
Якщо ти знаєш точну кількість вкладень, ти пишеш число (.flat(2), .flat(3), flat(4)...).
Якщо ти не знаєш, наскільки глибоко "заховані" дані, але хочеш зробити масив абсолютно плоским, використовується ключове слово Infinity.
Приклад:
const superDeep = [1, [2, [3, [4, [5]]]]];
superDeep.flat(Infinity); // [1, 2, 3, 4, 5]

📝 Завдання №27.1 (Обробка тегів та пошук):
Уяви, що ти отримуєш дані про пости в блозі, де кожен пост має масив тегів.
Створи масив allTags, який складається з декількох вкладених масивів з тегами:
[['js', 'react'], ['node', 'express'], ['js', 'html']]
Використай метод .flat(), щоб перетворити це на один плоский масив усіх тегів.
Використай метод .includes(), щоб перевірити, чи є серед усіх цих тегів тег 'react'.
Виведи результат перевірки (true/false) у консоль.

✅ Рішення:
const allTags = [
    ['js', 'react'], 
    ['node', 'express'], 
    ['js', 'html']
];

const flat = allTags.flat();
const includes = flat.includes('react');
console.log(flat);
console.log(includes);

📝 Завдання №27.2 (Глибоке розгладжування та пошук):
Створи масив deepArray: [1, [2, [3, 4]], 5].
Використай .flat(2), щоб повністю розгладити його до стану [1, 2, 3, 4, 5].
За допомогою .includes() перевір, чи є там число 3.
Виведи результат у консоль.

✅ Рішення:
const deepArray = [1, [2, [3, 4]], 5];

const flat = deepArray.flat(2);
const includes = flat.includes(3);

console.log(flat);
console.log(includes);

📝 Завдання №27.3 (Перевірка списку запрошених):
Це практична задача на комбінацію методів.
Створи масив guestList: ['Ivan', 'Anna', 'Oleg', 'Dmitro'].
Створи змінну newGuest = 'anna'.
Твоє завдання: перевірити, чи є цей гість у списку, але так, щоб регістр (велика чи мала літера) не мав значення.
Підказка: Спочатку створи новий масив, де всі імена будуть у нижньому регістрі (використай .map() та .toLowerCase()), а вже потім застосуй .includes().
Виведи результат (true/false) у консоль.

✅ Рішення:
const guestList = ['Ivan', 'Anna', 'Oleg', 'Dmitro'];
const newGuest = 'anna';

const checkGuests = guestList.map(guestName => guestName.toLowerCase());
const isThereAnna = checkGuests.includes(newGuest);

console.log(checkGuests);
console.log(isThereAnna);

📝 Завдання №27.4 (Глибоке очищення масиву):
Уяви, що ти розбираєш дуже заплутану структуру папок або категорій.
Створи масив chaos: [10, [20, [30, [40, [50, [60]]]]], 70].
Спробуй розгладити його двома способами:
Спершу на 3 рівні вглиб. Подивися в консолі, чи залишилися ще внутрішні масиви.
Потім розгладь цей же масив chaos за допомогою Infinity.
Після повного розгладжування (Infinity), використай метод .includes(60), щоб підтвердити, що число 60 тепер доступне на верхньому рівні.
Виведи результати обох етапів у консоль.

✅ Рішення:
const chaos = [10, [20, [30, [40, [50, [60]]]]], 70];
const thirdLevelFlat = chaos.flat(3);
console.log(thirdLevelFlat);

const infinityFlat = chaos.flat(Infinity);
const includes = infinityFlat.includes(60);

console.log(infinityFlat);
console.log(includes);
