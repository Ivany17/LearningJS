Урок №23: Пошук елемента з .find() 🎯 📍
Якщо .filter() — це сито, яке збирає групу елементів, то .find() — це снайпер, який шукає лише одну конкретну ціль.

1. Головна відмінність ⚡
.filter() завжди повертає масив (навіть якщо там один елемент або порожньо). 📦
.find() повертає перший конкретний елемент, який підійшов під умову. Якщо нічого не знайдено, він поверне undefined. 🔍

2. Як він працює? ⚙️
Метод перебирає масив зліва направо. Як тільки він знаходить перший елемент, для якого твоя умова повертає true, 
він зупиняє роботу і віддає цей елемент тобі.
const numbers = [5, 12, 8, 130, 44];
const found = numbers.find(element => element > 10);
// found буде 12, бо це перше число, що більше за 10.

3. Чому це зручно для Full-stack? 🛠️
У розробці ми часто шукаємо щось унікальне:
Знайти користувача за його ID. 🆔
Знайти товар за його артикулом. 🏷️
Знайти коментар за конкретним часом публікації. ⏱️

4. Порівняння з LINQ 🔍
У C# аналогом .find() є метод .FirstOrDefault(). Він так само шукає перший збіг 
і повертає значення за замовчуванням (null для об'єктів), якщо нічого не знайдено.

🏋️ Завдання №23.1: Пошук за ID (Level 1)
Уяви, що в тебе є список студентів, і тобі потрібно знайти інформацію про конкретну людину, знаючи її унікальний номер (ID).
Твоя умова:
Створи масив об'єктів students, які мають id і name
Використай метод .find(), щоб знайти студента з id: 102.
Збережи результат у змінну student і виведи її в консоль.
Використай той самий масив students.
Спробуй знайти студента з id: 999 (якого точно немає в списку).
Виведи результат у консоль.

✅ Рішення:
const students = [
    { id: 101, name: "Олександр" },
    { id: 102, name: "Марія" },
    { id: 103, name: "Дмитро" }
];

const student = students.find(n => n.id == 102);
const noneStudent = students.find(n => n.id == 999);
console.log(student);
console.log(noneStudent);

🏋️ Завдання №23.2: Пошук з деструктуризацією та логікою 🧩
Уяви, що ти пишеш систему для бібліотеки. Тобі потрібно знайти першу доступну книгу конкретного автора.
Створи масив, де будуть title, author, isAvailable
Твоя задача: Знайди першу книгу, у якої:
Автор — "Robert Martin". ✍️
Вона доступна (isAvailable має бути true). 📖
Вимога: Використай деструктуризацію прямо в параметрах методу .find(), щоб дістати author та isAvailable.

✅ Рішення:
const books = [
    { title: "Clean Code", author: "Robert Martin", isAvailable: false },
    { title: "Refactoring", author: "Martin Fowler", isAvailable: true },
    { title: "The Clean Coder", author: "Robert Martin", isAvailable: true },
    { title: "JavaScript: The Good Parts", author: "Douglas Crockford", isAvailable: true }
];

const certainBook = books.find(({author, isAvailable}) => author == "Robert Martin" && isAvailable);
console.log(certainBook);
