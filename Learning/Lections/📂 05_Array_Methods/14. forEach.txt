Тема №14: Метод .forEach() 🔄
.forEach() — це вбудований метод масивів, який дозволяє виконати певну функцію для кожного елемента масиву по черзі.

1. Навіщо він, якщо є for? 🤔
Звичайний цикл for змушує тебе вручну створювати лічильник (let i = 0), стежити за довжиною масиву (i < arr.length) та збільшувати індекс (i++).
.forEach() робить все це автоматично "під капотом". Твій код стає коротшим і легшим для читання.

2. Синтаксис ✍️
Метод приймає у себе функцію (найчастіше стрілочну). Ця функція автоматично отримує поточний елемент масиву на кожній ітерації.

const users = ["Іван", "Марія", "Олексій"];
// Читається: "Для кожного користувача (user) виведи його ім'я"
users.forEach(user => {
  console.log(user);
});

3. Додаткові параметри 🛠️
Функція всередині .forEach може приймати не тільки сам елемент, а й його індекс:

const colors = ["Red", "Green", "Blue"];
colors.forEach((color, index) => {
  console.log(`${index}: ${color}`);
});
// Виведе: 
// 0: Red
// 1: Green
// 2: Blue

4. Головне обмеження ⚠️
На відміну від звичайного циклу for, у .forEach() не можна використовувати команди break або continue. 
Він пройдеться по всьому масиву від початку до кінця, і його неможливо зупинити посередині. Якщо треба зупинитися — використовуй for.

📝 Завдання №14 (Практика автоматизму):
Створи масив prices = [100, 250, 50, 1000, 320].
Використай метод .forEach().
Для кожного елемента виведи в консоль повідомлення:
"Товар №[індекс+1] коштує [ціна] грн."
(Підказка: використовуй другий параметр індексу в стрілочній функції).
Створи змінну let totalSum = 0; перед початком перебору.
Всередині .forEach() додавай ціну кожного товару до totalSum.
Після завершення методу виведи в консоль фінальну суму всіх товарів: "Загальна вартість: [сума] грн."

✅ Рішення:
const prices = [100, 250, 50, 100, 320];

let totalSum = 0;
prices.forEach((price, index) => {
    console.log(`Товар №${index + 1} коштує ${price} грн.`);
    totalSum += price;
})

console.log(`Загальна вартість: ${totalSum} грн.`);

📝 Завдання №14.2: Модератор імен
Уяви, що користувачі вводили свої імена як заманеться, а тобі треба привести їх до ладу для списку.
Створи масив rawNames = ["іван", "мАрія", "олексій", "дмитро"].
Створи новий порожній масив cleanNames = [].
Використай .forEach(), щоб для кожного імені:
Зробити першу літеру великою, а решту маленькими. (Підказка: використовуй name[0].toUpperCase() + name.slice(1).toLowerCase()).
Додати виправлене ім'я в масив cleanNames.
Виведи cleanNames у консоль.

✅ Рішення:
const rawNames = ["іван", "мАрія", "олексій", "дмитро"];
const cleanNames = [];
rawNames.forEach(name => {
    let fixedName = name[0].toUpperCase() + name.slice(1).toLowerCase();
    cleanNames.push(fixedName);
});

console.log(cleanNames);

📝 Завдання №14.3: Аналітика температури
Тобі прийшли дані з датчиків за тиждень.
Масив: const weeklyTemp = [18, 22, 15, 30, 27, 10, 21].
Використай .forEach(), щоб проаналізувати кожен день:
Якщо температура вище 25 — виведи: "День №[індекс]: Спекотно ([температура])".
Якщо нижче 15 — виведи: "День №[індекс]: Холодно ([температура])".
В іншому випадку — виведи: "День №[індекс]: Комфортно".

✅ Рішення:
const weeklyTemp = [18, 22, 15, 30, 27, 10, 21];
weeklyTemp.forEach((temperature, index) => {
    if(temperature > 25){
        console.log(`День №${index + 1}: Спекотно (${temperature} °C)`);
    } else if(temperature < 15){
        console.log(`День №${index + 1}: Холодно (${temperature} °C)`);
    } else {
        console.log(`День №${index + 1}: Комфортно`)
    }
});

📝 Завдання №14.4: Генератор логів (Debug mode)
Часто розробники використовують .forEach() для швидкої перевірки стану даних.
Створи масив servers = ["Online", "Offline", "Online", "Maintenance", "Online"].
Використай .forEach() так, щоб вивести статус кожного сервера з його порядковим номером, але тільки якщо він не "Online".
Формат виводу: "[УВАГА] Сервер №[індекс] має статус: [статус]".

✅ Рішення:
const servers = ["Online", "Offline", "Online", "Maintenance", "Online"];
servers.forEach((server, index) =>{
    if(server !== "Online"){
        console.log(`[УВАГА] Сервер №${index + 1} має статус: ${server}`)
    }
});
