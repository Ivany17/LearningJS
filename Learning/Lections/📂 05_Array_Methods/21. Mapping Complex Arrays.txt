Урок №21: Advanced Data Transformation: Map, Destructuring, and Spread 🛠️ 🎓
У цьому уроці ми поєднуємо три потужні інструменти: метод .map(), деструктуризацію (Array/Object Destructuring) та оператор Spread.

1. Деструктуризація в параметрах функції 📥
Коли ми використовуємо .map(), функція-зворотний виклик (callback) приймає поточний елемент масиву. 
Якщо цей елемент сам є масивом або об'єктом, ми можемо деструктуризувати його прямо в дужках параметрів.
Приклад:
const points = [[10, 20], [30, 40]];
// Замість (point) => { const [x, y] = point; ... }
// Ми пишемо одразу:
const coords = points.map(([x, y]) => ({ x, y }));

2. Робота з вкладеними структурами 🪆
Якщо масив містить ще один масив всередині (як оцінки в нашому завданні), ми можемо дістати його під власною назвою:
const [name, age, grades] = student; — тут grades стане звичайним масивом, з яким можна працювати далі.

3. Поєднання зі Spread для оновлення даних ✨
Оператор Spread (...) дозволяє нам створити копію існуючого масиву або об'єкта, додаючи нові елементи без зміни оригіналу.
const allGrades = [...grades, 100]; — ми беремо всі оцінки зі старого списку і додаємо в кінець сотню.

Цей підхід використовується для перетворення масивів складних структур (наприклад, масивів масивів) у масиви об'єктів, 
що є стандартною практикою при роботі з API.
Деструктуризація в параметрах: Ми розбираємо кожен вкладений масив [name, age, grades] прямо на вході у функцію map.
Shorthand properties: Якщо назва ключа об'єкта збігається з назвою змінної (studentName, age), нам не потрібно писати age: age.
Глибоке копіювання масиву: Ми створюємо новий масив allGrades, розпаковуючи туди старі значення за допомогою ...grades і додаючи нове.

📝 Завдання №21.1: Об'єднана практика (Final Boss) 🏆
Це завдання — справжня перевірка всього, що ми вивчили за останні уроки: 
деструктуризацію, оператор Spread та метод .map(). Ми об'єднаємо ці теми, щоб перетворити "сирі" дані на зручну структуру.
Умова завдання:
Уяви, що ти отримуєш дані про студентів у вигляді масиву масивів. Кожен студент — це масив: [ім'я, вік, [оцінка1, оцінка2]].
Створи масив rawStudents:
[["Олег", 20, [85, 90]], ["Анна", 19, [95, 100]], ["Іван", 21, [70, 80]]]
Використай .map(), щоб створити новий масив studentProfiles.
Логіка всередині .map():
Використай деструктуризацію масиву прямо в параметрах функції, щоб отримати name, age та масив grades.
Поверни новий об'єкт для кожного студента.
В об'єкті мають бути властивості: name, age.
Додай властивість allGrades, куди за допомогою оператора Spread скопіюй усі оцінки з масиву grades 
і додай наприкінці ще одну оцінку — 100 (бо всі студенти молодці). 🎓
Виведи studentProfiles у консоль.

✅ Рішення:
/** @type {[string, number, number[]][]} */ // - [string, number, number[]] — кожен елемент цього масиву є масивом (кортежем), що містить рядок, число та масив чисел.
const rawStudents = [["Олег", 20, [85, 90]], ["Анна", 19, [95, 100]], ["Іван", 21, [70, 80]]];

const studentProfiles = rawStudents.map(
    ([studentName, age, grades]) => ({
        studentName, 
        age, 
        allGrades: [...grades, 100]
    })
);

console.log(studentProfiles);

📝 Завдання №21.2: Трансформація списку товарів 🛒
Уяви, що ми отримуємо дані про товари зі складу у вигляді масиву масивів. Кожен товар описаний так: [назва, ціна, [категорія1, категорія2]].
Масив даних:
const inventory = [["Smartphone", 500, ["tech", "mobile"]], ["Laptop", 1200, ["tech", "work"]]];
Твоя задача: за допомогою .map() створити масив об'єктів productCards.
Вимоги до об'єкта:
Використай деструктуризацію в параметрах map.
Об'єкт має містити: name, price.
Властивість allTags: має містити всі старі категорії (через Spread) + новий тег "sale".
Додай властивість currency: завжди рядок "USD".

✅ Рішення:
/** @type {[string, number, string[]][]} */
const inventory = [["Smartphone", 500, ["tech", "mobile"]], ["Laptop", 1200, ["tech", "work"]]];
const productCards = inventory.map(
    ([gadget, price, categories]) => ({
        gadget,
        price,
        allTags: [...categories, "sale"],
        currency: "USD"
    })
);
console.log(productCards);

📝 Завдання №21.3: Трансформація результатів марафону 🏃‍♂️
Уяви, що ми маємо дані про забіг. Кожен бігун представлений масивом: [ім'я, час_у_хвилинах, [місто, країна]].
Масив даних:
const raceResults = [["Petro", 125, ["Kyiv", "Ukraine"]], ["Jane", 140, ["London", "UK"]]];
Твоя задача: за допомогою .map() створити масив об'єктів runnerCards.
Вимоги до об'єкта:
Використай деструктуризацію масиву в параметрах map.
В об'єкті мають бути властивості: name та time.
Властивість location: має бути об'єктом, створеним з масиву [місто, країна] (назви ключі як city та country).
Додай властивість status: якщо час менше 130 хвилин — рядок "fast", інакше — "normal". (Тут можна використати тернарний оператор з Теми №7).

✅ Рішення:
/** @type {[string, number, string[]][]} */
const raceResults = [["Petro", 125, ["Kyiv", "Ukraine"]], ["Jane", 140, ["London", "UK"]]];
const runnerCards = raceResults.map(
    ([runnerName, time, location]) => ({
        runnerName,
        time,
        locationName: {city : location[0], country : location[1]},
        status: time < 130 ? "fast" : "normal",
    })
);

console.log(runnerCards);

// АБО ЩЕ ОДИН ВАРІАНТ ЗІ ЗМІНОЮ масиву

/** @type {[string, number, string[]][]} */
const raceResults = [["Petro", 125, ["Kyiv", "Ukraine"]], ["Jane", 140, ["London", "UK"]]];
const runnerCards = raceResults.map(
    ([runnerName, time, [city, country]]) => ({ // ми змінили масив тут
        runnerName,
        time,
        locationName: {city : city, country : country}, // і тепер використовуємо city і country тут
        status: time < 130 ? "fast" : "normal",
    })
);

console.log(runnerCards);
