Урок №22: Фільтрація масивів з .filter() 🔍 🛡️
Метод .filter() дозволяє створити новий масив, у який потраплять лише ті елементи оригінального масиву, 
що пройшли "перевірку" (відповідають твоїй умові).

1. Основна ідея 💡
Це як сито: ти засипаєш масив, ставиш сітку (умову), і на виході отримуєш лише те, що не провалилося крізь отвори. 🕸️
Оригінальний масив ніколи не змінюється (принцип імутабельності). 💎
Якщо жоден елемент не підійшов — ти отримаєш порожній масив []. 🕳️

2. Як це працює технічно ⚙️
Метод приймає функцію-зворотного виклику (callback), яка має повертати true (залишити елемент) або false (викинути).
const newArray = array.filter((element) => {
  return element > 10; // Умова: якщо так, елемент іде в новий масив
});

3. Стрілочний запис (LINQ style) 🏹
Оскільки ти знаєш C#, тобі найзручніше буде писати це в один рядок:
const result = numbers.filter(n => n > 15); 🧩
Це ідентично .Where(n => n > 15) у LINQ.

4. Коли використовувати? 🚦
Видалити товар із кошика. 🗑️
Показати лише повідомлення від друзів. ✉️
Відфільтрувати результати пошуку за ціною. 💰

🏋️ Завдання №22.1: Розігрів (Level 1)
Уяви, що ти розробляєш систему для входу в клуб. У тебе є масив віку відвідувачів, які стоять у черзі. 
Тобі потрібно пропустити лише тих, кому вже є 18 років.
Твоя умова:
Створи масив ages із числами: [14, 18, 22, 16, 35, 12, 21].
Використай метод .filter(), щоб створити новий масив adults.
Виведи масив adults у консоль.

✅ Рішення:
const ages = [14, 18, 22, 16, 35, 12, 21];
const adults = ages.filter(age => age >= 18);
console.log(adults);

🏋️ Завдання №22.2: Фільтрація каталогу товарів 🛒
Уяви, що ти створюєш функціонал для інтернет-магазину. У тебе є список смартфонів, 
і користувач хоче побачити лише ті, що коштують менше певного бюджету.
Створи масив products, де будуть name та price.
Твоя задача: Використай .filter(), щоб створити масив affordablePhones, куди потраплять лише товари, ціна яких менше або дорівнює 750.
Підказка: Кожен елемент масиву тепер — це об'єкт, тож звертайся до ціни через крапку (наприклад, product.price).

✅ Рішення:
const products = [
    { name: "iPhone 15", price: 900 },
    { name: "Samsung S23", price: 800 },
    { name: "Xiaomi 13", price: 500 },
    { name: "Pixel 8", price: 700 }
];

const affordablePhones = products.filter(product => product.price <= 750);
// або const affordablePhones = products.filter(({ price }) => price <= 750);
console.log(affordablePhones);

🏋️ Завдання №22.3: Комплексна перевірка (Final Boss) 🏆
Давай закріпимо тему на більш складному прикладі, де потрібно перевірити відразу дві умови.
Уяви, що ти фільтруєш список користувачів для розсилки новин.
Створи масив,де будуть name, age, isActive
Твоя задача: Створи масив targetUsers, до якого потраплять лише ті, хто:
Має статус активний (isActive дорівнює true). ✅
Вже є повнолітнім (age більше або дорівнює 18). 🔞

✅ Рішення:
const users = [
    { name: "Oleg", age: 30, isActive: true },
    { name: "Anna", age: 17, isActive: true },
    { name: "Ivan", age: 25, isActive: false },
    { name: "Maria", age: 20, isActive: true }
];

const targetUsers = users.filter(user => user.age >= 18 && user.isActive == true);
// або const targetUsers = users.filter(({ age, isActive }) => age >= 18 && isActive);
console.log(targetUsers);
