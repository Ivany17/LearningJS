Урок №24: Метод .reduce() — Математичний підхід 🧮
Метод .reduce() — це справжній «швейцарський ніж» для роботи з масивами. Якщо .map() чи .filter() повертають новий масив, 
то .reduce() допомагає перетворити цілий список даних на одне значення (число, рядок або навіть об'єкт).

Уяви, що в тебе є коробка з числами, і тобі потрібно знайти їхню суму. Ти береш перше число, додаєш до нього друге, 
запам’ятовуєш результат, додаєш третє і так далі. Саме це робить .reduce().

1. Основні поняття 🧩
Метод приймає функцію, яка має два головні аргументи:
Акумулятор (accumulator) 📥: це твоя «пам'ять». Тут зберігається результат попереднього кроку.
Поточне значення (currentValue) 🔢: елемент масиву, який ми обробляємо прямо зараз.

Також у .reduce() є початкове значення — це число, з якого акумулятор починає свою роботу.

2. Як це виглядає в коді? 💻
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((accumulator, currentValue) => {
  return accumulator + currentValue;
}, 0); // 0 — це початкове значення

3. Приклад коду для порівняння C# і JavaScript 💻
Уяви, що ми хочемо порахувати суму чисел [1, 2, 3].
У C# (LINQ):
var numbers = new int[] { 1, 2, 3 };
var sum = numbers.Aggregate(0, (acc, next) => acc + next);

У JavaScript:
const numbers = [1, 2, 3];
const sum = numbers.reduce((acc, curr) => acc + curr, 0);

Головна відмінність: У JS початкове значення (0) ставиться після функції-колбеку, а в C# — перед нею.

4. Групування даних (Grouping) 🗄️
Головна відмінність тут у початковому значенні. Якщо для суми ми використовували 0, а для рядка "", 
то для групування ми використовуємо порожній об'єкт {}. 📦

Як це працює? ⚙️
На кожному кроці ми:
Беремо категорію поточного товару. 🏷️
Перевіряємо, чи є вже такий "кошик" (ключ) у нашому об'єкті-акумуляторі.
Якщо немає — створюємо порожній масив.
Додаємо товар у відповідний масив.

Повертаємо оновлений акумулятор для наступного кроку.
Приклад логіки 🧩
const result = items.reduce((acc, item) => {
  const category = item.category;
  
  if (!acc[category]) {
    acc[category] = []; // Створюємо категорію, якщо її ще немає
  }
  
  acc[category].push(item); // Додаємо товар у потрібний кошик
  return acc;
}, {}); // Стартуємо з порожнього об'єкта

🏋️ Завдання №24.1: Рахунок у магазині

Використай метод .reduce(), щоб додати всі ціни одна до одної.

🏋️ Завдання №24.2: Reduce та об'єкти (Level Up) 🚀
Ускладнимо задачу. Зазвичай у масивах ми маємо не просто числа, а об'єкти. Давай знайдемо загальну вартість товарів, враховуючи їхню кількість.

Використай .reduce(), щоб порахувати загальну суму всього кошика.

Не забудь про початкове значення 0.

✅ Рішення:

    { name: "Apple", price: 20, quantity: 5 },
    { name: "Banana", price: 10, quantity: 10 },
    { name: "Orange", price: 30, quantity: 2 }
];

const totalSum = cart.reduce((acc, n) => { return acc + n.price * n.quantity}, 0);
console.log(totalSum);

🏋️ Завдання №24.3: Максимальне число (Level: Thinker) 🧠
Метод .reduce() можна використовувати не тільки для сумування. Давай спробуємо знайти найбільше число в масиві.
Створи масив numbers з числами.
Твоя задача:
Використай .reduce(), щоб знайти максимальне число.
В акумуляторі (acc) на кожному кроці має залишатися те число, яке є більшим: або те, що вже лежить в acc, або поточне n.
Для порівняння можна використати звичайний if або Math.max(acc, n).
Початкове значення візьми перше число масиву або 0.

✅ Рішення:
const numbers = [15, 42, 7, 89, 23];
const maxNumber = numbers.reduce((acc, n) => {
    if(acc > n){
        return acc;
    } else {
        return n;
    }
}, numbers[0]);

console.log(maxNumber);

🏋️ Завдання №24.7: Підрахунок сум за категоріями
Це завдання поєднує математику та групування. Уяви, що тобі потрібно не просто згрупувати товари, 
а відразу порахувати, скільки грошей припадає на кожну категорію.
Створи масив transactions, де будуть id, category (Їжа, Транспорт, Розваги) і сума витрачених грошей amount. 
Твоя задача:
Використай .reduce(), щоб створити об'єкт.
Ключами мають бути категорії ("Їжа", "Транспорт", "Розваги").
Значенням кожного ключа має бути число — сума всіх amount для цієї категорії.
Початкове значення — {}.

✅ Рішення:
const transactions = [
    { id: 1, category: "Їжа", amount: 200 },
    { id: 2, category: "Транспорт", amount: 50 },
    { id: 3, category: "Їжа", amount: 150 },
    { id: 4, category: "Розваги", amount: 300 },
    { id: 5, category: "Транспорт", amount: 100 }
];

const groupByCategory = transactions.reduce((acc, n) => {
    const category = n.category;

    if(!acc[category]){
        acc[category] = n.amount;
    } else {
        acc[category] += n.amount;
    }

    return acc;
}, {});
// незважаючи на те, що рядки підсвічуються червоним кольором, цей код все одно можна запустити і він буде працювати
console.log(groupByCategory);

Твій масив:

Твоя задача:
Використай .reduce().
Ключами мають бути числа (довжина слова).
Значеннями — масиви слів цієї довжини.

},{});
