Тема №6: Шаблонні рядки (Template Literals) 📜✨
Це сучасний спосіб роботи з текстом, який з'явився у JavaScript (ES6) і практично витіснив старий метод склеювання рядків через плюс +.

1. Зворотні лапки (Backticks) ⌨️
Для створення шаблонного рядка використовуються не звичайні лапки, а зворотні — ` ` (вони знаходяться на клавіші з тильдою ~ під клавішею Esc).

2. Інтерполяція (Вставка змінних) 📥
Замість того, щоб розривати текст: "Привіт " + name + "!", ми просто пишемо змінні всередині рядка, використовуючи конструкцію ${...}.
Приклад:
let name = "Іван";
console.log(`Привіт, ${name}!`); // Виведе: Привіт, Іван!

3. Багаторядковість 📑
У звичайних лапках не можна просто натиснути Enter — буде помилка. А в шаблонних рядках ти можеш писати текст на кілька рядків так само як у звичайному блокноті.

Приклад:
let list = `
1. Купити хліб
2. Купити молоко`;

4. Вирази всередині ${} 🧮
Всередині фігурних дужок можна не тільки писати назву змінної, а й виконувати математику або логіку.

Приклад: ${2 + 2} виведе 4.

📝 Завдання №6 (Template Literals):
Давай створимо звіт про замовлення в інтернет-магазині, використовуючи всі твої знання (типи даних, математику та шаблонні рядки).
Створи змінні: item = "Смартфон", price = 800, quantity = 2.
Створи змінну totalMessage за допомогою зворотних лапок. Текст має бути багаторядковим (використовуй Enter у коді):
Перший рядок: Товар: [назва товару]
Другий рядок: Загальна вартість: [результат множення ціни на кількість]$
Третій рядок: Преміум замовлення: [результат перевірки, чи загальна сума більша за 1000]
Виведи totalMessage у консоль.

✅ Рішення:
let item = "Смартфон";
let price = 800;
let quantity = 2;

let totalMessage = `Товар: ${item}
Загальна вартість: ${price * quantity}$
Преміум замовлення: ${price * quantity > 1000}`
console.log(totalMessage);

📝 Завдання №6.2 (Профіль користувача):
Уяви, що ти створюєш картку користувача для соціальної мережі.
Створи три змінні:
username (будь-яке ім'я);
postsCount (кількість постів, наприклад 15);
isOnline (булеве значення true або false).
Створи змінну userCard, використовуючи зворотні лапки. Текст має бути таким (з переносами рядків):
Користувач: @[username]
Активність: [кількість постів] постів
Статус: [тут має бути результат логічного порівняння isOnline === true]
Виведи userCard у консоль.
